# FizzBuzz

Mostrar Números de 1 a 100, de modo que
Se for divisível por 3 mostre a palavra Fizz
Se for divisível por 5 mostre a palavra Buzz
Se for divisível por ambos mostre FizBuzz

```bash
dotnet run
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
16
17
...
```
